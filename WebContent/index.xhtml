DESCRIÇÃO

O Nemo Ontology Mapping (NOM) é uma aplicação para visualização e consultas SPARQL de dados governamentais sobre servidores e organizações mapeados com a Ontologia de Organizações. Esses dados são apresentados em RDF, onde cada recurso é representado por uma URI (em forma de URL) ou Literal. Com isso, é possível derreferenciar as URIs, ou seja, obter algum resultado quando alguma URI é pesquisada.

FORMAS DE USO

Há quatro forma de derreferenciar as URIs que são apresentas abaixo:

  1. URI : Trás o resultado em tabela.
  2. URI + ".rdf" : Trás o resultado em RDF.
  3. URI + ".html" : Trás o resultado em uma visualização mais agradável.
  4. URI + ".graph" : Trás o resultado em Grafo.

SPARQL ENDPOINT

Duas formas de realizar consulta SPARQL são disponíveis. A primeira é o SPARQL Query Editor que permite realizar consultas simples (em código) e a segunda é um SPARQL interativo (iSPARQL) no qual é possível realizar consultas através de grafos visuais.

  • SPARQL Query Editor
  • iSPARQL

EXEMPLOS

Abaixo são apresentados exemplos das formas de derreferenciar uma URI. Em seguida, é apresentado o resultado ao derreferenciar a primeira das quatro formas possíveis.

  2. RDF: http://localhost:8080/NOM/resource/person/1682175.rdf
  3. HTML: http://localhost:8080/NOM/resource/person/1682175.html
  4. GRAPH: http://localhost:8080/NOM/resource/person/1682175.graph
